# ロケハンダッシュボード (PWA)

星と光の時間割 — 日没・月・天の川・雲・潮汐を1画面で。

## 構成
- `index.html` — アプリ本体(単体でも動作。天文計算は完全オフライン)
- `sw.js` — オフラインキャッシュ(アプリ本体+最後に取得した予報)
- `manifest.webmanifest` / `icons/` — ホーム画面用

## 公開手順 (GitHub Pages・約5分)
1. GitHubで新規リポジトリを作成(例: `rokehan`)
2. このフォルダの中身をすべてアップロード(Add file → Upload files)
3. Settings → Pages → Branch を `main` / `(root)` にして Save
4. 数分後に `https://<ユーザー名>.github.io/rokehan/` で公開される

## スマホでアプリ化
- iPhone: Safariで開く → 共有 → **ホーム画面に追加**
- Android: Chromeで開く → メニュー → **アプリをインストール**

## オフライン挙動
- 天文計算(太陽・月・天の川)は常にオフラインで動作
- 雲量・潮位は、直近にオンラインで開いたときの予報をキャッシュ表示
  (圏外の山に入る前に一度開いておくと、そのデータを持ち歩けます)

## 注意
- 潮位は海洋モデル(Open-Meteo Marine)の予測値。干潟へ渡る最終判断は
  気象庁の潮位表(那智勝浦エリアは「浦神」)で必ずダブルチェックを。
